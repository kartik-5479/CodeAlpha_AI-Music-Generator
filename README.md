# 🧠 How It Works

```text
        MIDI Dataset
              │
              ▼
      Data Preprocessing
      (music21 Library)
              │
              ▼
     Sequence Generation
              │
              ▼
      LSTM Neural Network
              │
              ▼
        Music Prediction
              │
              ▼
     Generate MIDI File
              │
              ▼
   Play • Download • Save
```
